## 라이브 vs 추출
* 라이브 : DB에 직접적으로 쿼리를 날림
  - DB가 실시간으로 업데이트 될 때 내가 만들어놓은 화면도 실시간으로 업데이트 됨
* 추출 : 테이블의 스캔본이라고 생각하면 됨
  - DB가 실시간으로 업데이트 될 때 업데이트 내용을 반영하지 못함. 반영하려면 추출을 다시 해야함.
